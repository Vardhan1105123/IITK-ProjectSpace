# IITK ProjectSpace

This full-stack web application is made as a course project of **CS253: Software Development and Operations** under the mentorship of TA Seetaramayya Lavu.



Currently, projects developed by the IITK Community reside in isolated GitHub repositories, personal devices, and club rooms, while recruitments often rely on scattered Google Forms or cold mailing. IITK ProjectSpace is an integrated platform that seeks to solve this fragmentation by creating a centralized, self-contained web-based application. Users can seamlessly manage portfolios, showcase innovative projects, find collaborators, and streamline the recruitment workflow.

The platform simplifies project discovery and team building with the following salient features:

**For Project Creators & Recruiters:**
* List projects with detailed Markdown descriptions, domains, and media uploads.
* Open recruitment posts specifying prerequisites, allowed departments (e.g., CSE, EE, BSBE), and designations (e.g., UG, PG, PhD).
* Manage and track incoming applications (Accept/Reject) with an integrated dashboard.
* Receive comments and feedback on projects.

**For Students & Applicants:**
* Search projects and recruitments efficiently using global semantic and full-text search.
* Apply directly to open recruitments within the platform.
* Receive real-time notifications for application updates, team invitations, and post comments.

The interface of the web app is intuitive and restricted strictly to the campus community via `@iitk.ac.in` OTP email verification, ensuring a secure and seamless experience.
